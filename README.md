##  프로젝트 소개

이 프로젝트는 기본적인 4원소(불🔥, 물💧, 땅🟫, 공기🌬️)에서 시작하여, 사용자의 상상력과 AI의 추론 능력을 통해 새로운 단어와 개념을 확장해 나가는 것을 목표로 합니다. 사용자가 두 요소를 합치면, 시스템은 두 단어의 맥락을 분석하여 가장 알맞은 새로운 단어와 그에 어울리는 이모지를 생성하여 반환합니다.

단순한 물질의 결합뿐만 아니라 추상적인 개념, 인물, 현상 등 다양한 범주의 명사를 만들어낼 수 있습니다.

##  주요 기능

* **무한한 조합 시스템 (Generative Crafting)**
    * 정해진 조합법이 없습니다. AI 모델이 두 단어의 관계를 분석하여 창의적이고 논리적인 결과를 실시간으로 생성합니다.
* **드래그 앤 드롭 인터페이스**
    * 직관적인 UI를 통해 요소를 자유롭게 배치하고, 드래그하여 다른 요소 위에 올려놓는 것만으로 합성을 시도할 수 있습니다.
* **이모지 자동 생성**
    * 모든 생성된 단어에는 AI가 선정한 가장 적절한 이모지가 함께 부여되어 시각적인 재미를 더합니다.
* **지능형 캐싱 (Caching)**
    * 한 번 생성된 조합법은 저장되어, 다음번 동일한 조합 시도 시 기다림 없이 즉시 결과를 확인할 수 있습니다.

## 🎮플레이 방법

1.  **요소 선택:** 우측 사이드바(Select Spot)에 있는 기본 요소 중 원하는 것을 메인 화면으로 드래그합니다.
2.  **조합하기:** 메인 화면(Drag Spot)에 있는 요소를 드래그하여 다른 요소 위에 겹쳐 놓습니다.
3.  **결과 확인:** 잠시 생각하는 시간(Loading ⏳)이 지나면, 두 요소가 합쳐져 새로운 요소가 탄생합니다.
4.  **확장하기:** 새로 만들어진 요소를 다시 다른 요소와 결합하여 도감에 없는 새로운 발견을 계속해 나가세요.
